**Цель проекта:** Автоматизация процесса сбора и обработки сообщений о нарушениях для сокращения времени между обнаружением и фиксацией инцидентов.

**Используемые технологии:** Python, SQL, python-telegram-bot, BPMN, Use Cases

**Процесс анализа:**

1. **Анализ предметной области:** Изучение существующего процесса регистрации нарушений

3. **Проектирование данных:** Разработка структуры базы данных для хранения заявок
4. **Описание сценариев:** Формализация пользовательских сценариев через Use Cases
5. **Прототипирование:** Создание макета диалога бота и логики взаимодействия

**Результаты:**
- Разработана структурированная система сбора нарушений
- Создана документация для дальнейшей разработки
- Определены ключевые метрики эффективности системы

### Система учета рабочего времени 

**Цель проекта:** Автоматизация учета рабочего времени сотрудников с интеграцией в мессенджер для повышения точности и прозрачности отчетности.

**Используемые технологии:** Python, aiogram, SQL, UML, методологии сбора требований

**Процесс анализа:**

1. **Опрос стейкхолдеров:** Выявление потребностей сотрудников и руководителей
2. **Анализ бизнес-процессов:** Исследование текущего workflow учета времени
3. **Формулировка требований:** Разделение на функциональные и нефункциональные требования
4. **Проектирование API:** Определение методов взаимодействия (/start_work, /end_work, /get_report)

**Результаты:**
- Оптимизирован процесс учета рабочего времени
- Разработана архитектура системы с четким разделением ответственности
- Создана техническая документация для реализации
